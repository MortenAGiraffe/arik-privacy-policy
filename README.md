# ARIK GPT Actions Privacy Policy

## Summary
ARIK is a custom GPT that may use external Actions to retrieve information or perform tasks outside ChatGPT, only when requested by the user.

## What data is accessed
- The GPT may access GitHub repository contents via the GitHub API if a user asks it to read a repository.
- Access is limited to file listings and specific file contents needed to answer the request.

## Authentication
- If GitHub access is enabled, the GPT uses an API credential stored as a secret in the GPT Actions configuration.
- The credential is not shown to users.

## How data is used
- Retrieved content is used only to generate responses inside the chat.
- No additional storage, resale, or sharing of repository data is performed by the GPT outside the chat experience.

## User responsibilities
- Do not paste secrets (API keys, passwords, private tokens) into chat.
- If sensitive material must be referenced, redact it before sharing.
